# Shop-Shop

## Description

A demo ecommerce site using React and Redux

## Made with
- React
- Redux
- GraphQL
- Stripe
- Apollo server
- JSON Web Tokens
- Node
- Express
- MongoDB
- Mongoose


## Installation

To install necessary dependencies, run the following command:

```
npm i
```

## Usage

Run the development build by using
```
npm start
```
